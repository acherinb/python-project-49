# Brain games

This repository is made by Andrey Cheredinov, a student of Hexlet educational project.

### Requirements
Python 3.10 or above

### Installation
Use following commands for installing:

	poetry install
	poetry build
	poetry publish
	python3 -m pip install dist/*.whl
